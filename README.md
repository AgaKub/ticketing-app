# Ticketing App

Simple ticketing platform for small event promoters.

## Goal
Provide an affordable ticketing system for small gigs and events without high commissions from large ticketing platforms.

## Core Features (MVP)
- Create events
- Sell tickets online
- Generate QR-code tickets
- Scan tickets at event entrance
- Track ticket sales

## Target Users
Small promoters organizing:
- club gigs
- pub concerts
- stand-up shows
- small festivals

## Future Features
- Promo codes
- Sales analytics
- Capacity monitoring
- Marketing integrations

## Planned Tech Stack
- Python
- Django
- PostgreSQL
- Stripe
- QR code library
